# Vulcano Marketplace

Plugin marketplace de Claude/Cowork. Contiene el plugin **vulcano** con todas
las skills de Luis (25 skills), sincronizadas desde /root/.claude/skills.

## Agregar en Cowork / Claude Code

    /plugin marketplace add turbillon50/vulcano-marketplace
    /plugin install vulcano@vulcano-marketplace

## Sync (en Hetzner)

    /root/brain/sync-marketplace.sh

Version del plugin: 0.1.14

Nota: secrets se redactan automaticamente antes de publicar (repo publico).
